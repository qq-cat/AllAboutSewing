# Clo3D — Basics

## What is Clo3D?

Clo3D is a 3D garment simulation software used by designers to visualize how fabric drapes and behaves on a virtual avatar before making physical samples.

---

## Interface Overview

_(Add screenshot or sketch: `![interface overview](assets/interface-overview.png)`)_

Key panels:
- **2D Pattern Window** — where you draw and edit flat patterns
- **3D Garment Window** — live simulation view
- **Object Browser** — manage garment pieces, avatars, and fabrics
- **Property Editor** — adjust fabric physics, colors, and settings
