# Clo3D — Draping & Fabric Simulation

## Key Concepts

- **Particle distance** — controls fabric resolution; lower = more detailed but slower
- **Fabric physics** — tension, bending, gravity affect how fabric hangs
- **Collision** — how the garment interacts with the avatar body

---

## Draping Workflow

1. Arrange 2D pattern pieces around the avatar in 3D view
2. Sew seams in 2D window
3. Hit `Space` to simulate
4. Adjust fabric properties as needed

## Fabric Property Settings

| Property | What it does | Notes |
|---|---|---|
| Bending | Stiffness of fabric fold | High = denim-like |
| Tension | Stretch resistance | Low = jersey |
